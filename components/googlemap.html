Google Map

Conference uses Google map with Snazzy Maps, with different color schemes aimed towards web designers and developers.

with steps and elevators to observation decks.

Javascript

Get your API key⟶

<script src="https:maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap" async></script>
Map color schemes

Change the value of data-theme='' to any of the followings

  • Default
  • Gray
  • Midnight
  • Hopper
  • Beard
  • AssassianCreed
  • SubtleGray
  • Tripitty
